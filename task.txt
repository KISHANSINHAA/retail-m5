RetailSense AI: End-to-End Retail Analytics & Sales Forecasting Platform using Azure Databricks, PySpark, Delta Lake, and Grok LLM

Project Objective

Build a cloud-based retail analytics platform that processes the M5 Forecasting Dataset using PySpark on Azure Databricks, implements a Medallion Architecture (Bronze, Silver, Gold) with Delta Lake, performs sales forecasting, visualizes business KPIs through Databricks Dashboards, and integrates the Grok LLM API to provide AI-powered natural language insights and business recommendations.

Complete Project Flow
Phase 1: Data Collection
Download the M5 Forecasting dataset.
Upload all CSV files to Azure Blob Storage.

Input Files

sales_train_validation.csv
calendar.csv
sell_prices.csv
Phase 2: Data Ingestion (Bronze Layer)

Using PySpark:

Read CSV files
Infer or define schema
Validate schema
Store raw data as Delta tables

Output

bronze_sales
bronze_calendar
bronze_prices
Phase 3: Data Cleaning (Silver Layer)

Perform:

Remove duplicate records
Handle null values
Convert data types
Standardize date formats
Validate IDs
Apply basic quality checks

Output

silver_calendar
silver_prices

Phase 10: AI Retail Assistant (Grok LLM)

Instead of sending raw data, the FastAPI backend:

Queries Gold tables.
Builds a concise KPI summary.
Sends only the summary plus the user's question to the Grok API.

Example capabilities:

Explain sales trends.
Identify top-performing categories.
Summarize monthly performance.
Recommend inventory actions.
Compare store performance.
Explain forecast results.

This minimizes API token usage and keeps responses fast.

Phase 11: Deployment

Deploy:

Data processing and dashboards on Azure Databricks.
FastAPI backend (Azure App Service, Azure Container Apps, or another low-cost hosting option).
Connect the backend securely to the Grok API using environment variables or Azure Key Vault (optional).
